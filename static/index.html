Syncing the UI with the Detection Engine

Project Description

We currently have a detection engine that operates based on instructions found in YAML files. These instructions may include a whitelist or a blacklist of organizations. This guides the detection engine to decide which detector it should run for a particular organization.

Simultaneously, our UI shows each organization which detectors are currently in operation for them. However, there's an issue: the UI doesn't always correctly show the information from the detection engine's configuration. In some cases, this could result in the UI displaying a detector as operating when it's not, due to being blacklisted in the detection engine's configuration.

Your task is to find a way to make sure the detection engine's configuration and the UI are correctly synced. We want the UI to accurately show the status of the detectors for each organization, based on the settings from the detection engine's configuration.
